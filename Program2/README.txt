If the executable is not there compile by using the makefile by typing "make"

Once the executable is there 
To run it do "./cpath <INPUT-FILE> <TARGET> <START> <DEST> <BUDGET>"

where <INPUT-FILE> is a string of a .txt file including the extension


ex:

./cpath g.txt 0 6 12
